# Flow iOS 사전 과제

## 과제 설명

- 앨범 리스트 및 앨범 내 사진 및 정보를 확인할 수 있는 앱입니다.
- 앱 실행 시 앨범 목록이 표시됩니다.
- 앨범을 선택하면 앨범 내 사진목록이 표시됩니다.
- 사진을 선택하면 사진의 원본 파일 이름과, 용량이 표시됩니다.



## 개발환경


- 기반 환경
  - OS: Mac OS X 11.4
  - IDE: Xcode 12.5.1
  - Swift 5.4
  - Git
- Application
  - UIKit
  - RxSwift
  - Then
  - Photos
